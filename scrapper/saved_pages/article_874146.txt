class="article-formatted-body article-formatted-body article-formatted-body_version-2"><div xmlns="http://www.w3.org/1999/xhtml"><p>Команда <a href="https://t.me/+QmrKn0wA8CdkNjAy">Spring АйО</a> рада поделиться с вами свежей новостью от JetBrains: стала доступна первая EAP-версия IntelliJ IDEA 2025.1. </p><p>Среди нововведений поддержка Java 24, улучшения отладчика и инструментов сборки.</p><p>Стартовала программа раннего доступа (EAP) для IntelliJ IDEA 2025.1! Уже <a href="https://www.jetbrains.com/idea/nextversion/"><u>доступна первая сборка</u></a>, в которой вы можете протестировать новые функции и улучшения, подготовленные для следующего крупного релиза.</p><details class="spoiler"><summary>Уточнение</summary><div class="spoiler__content"><p><em>Скачать EAP возможно только c не российских IP-адресов</em></p></div></details><figure class="full-width"><img data-src="https://habrastorage.org/getpro/habr/upload_files/56a/98a/a30/56a98aa300327180d8e3117a804f6d63.png" height="1440" src="https://habrastorage.org/r/w1560/getpro/habr/upload_files/56a/98a/a30/56a98aa300327180d8e3117a804f6d63.png" width="2560"/></figure><h2>Java</h2><h3>Ранняя поддержка Java 24</h3><p>К использованию уже доступна Java 24! Станьте одним из первых, кто испробует её новые возможности в IntelliJ IDEA. Просто настройте ваш SDK на Oracle OpenJDK 24 и измените уровень языка (Language Level) в разделе Project Structure | Project Settings | Project.</p><figure class="full-width"><img data-src="https://habrastorage.org/getpro/habr/upload_files/50f/fde/b0b/50ffdeb0b0ea1a3ff22a12c41d3f2a84.png" height="600" src="https://habrastorage.org/r/w1560/getpro/habr/upload_files/50f/fde/b0b/50ffdeb0b0ea1a3ff22a12c41d3f2a84.png" width="1500"/></figure><h2>Фреймворки и технологии</h2><h3>Более быстрый способ создания Spring Beans</h3><p>Начиная с этого EAP, вы можете создавать Spring Beans прямо из меню <em>New…</em> или <em>Generate…</em>. Больше нет необходимости создавать пустой класс и вручную добавлять аннотации – просто выберите нужную опцию, и всё готово!</p><figure class="full-width"><img data-src="https://habrastorage.org/getpro/habr/upload_files/2bf/349/25c/2bf34925c8985c50bfca2c2b536092ce.gif" height="600" src="https://habrastorage.org/getpro/habr/upload_files/2bf/349/25c/2bf34925c8985c50bfca2c2b536092ce.gif" width="1500"/></figure><p>Обязательно опробуйте это улучшение, чтобы упростить работу со Spring и сэкономить время.</p><h2>Инструменты сборки</h2><h3>Поддержка Gradle Daemon toolchains</h3><p>Начиная с Gradle 8.13, вы можете задавать определенную JVM для Gradle Daemon с <a href="https://docs.gradle.org/current/userguide/gradle_daemon.html#sec:daemon_jvm_criteria"><u>помощью нативного Gradle toolchain</u></a>. Ранее это было возможно только для самого проекта. Теперь IDE синхронизируется с конфигурацией Gradle и даже позволяет Gradle автоматически загружать нужную JVM, если это необходимо.</p><p>Выбор настроек в <em>Preferences/Settings | Build Tools | Gradle</em> остаётся полностью синхронизированным с конфигурацией Gradle. Хотя эта настройка является опциональной, мы рекомендуем её использовать, чтобы избежать ошибок JVM Daemon и обеспечить плавную синхронизацию проектов.</p><figure class="full-width"><img data-src="https://habrastorage.org/getpro/habr/upload_files/f5f/0d3/8e6/f5f0d38e60e174b46ea6db43dde66826.png" height="600" src="https://habrastorage.org/r/w1560/getpro/habr/upload_files/f5f/0d3/8e6/f5f0d38e60e174b46ea6db43dde66826.png" width="1500"/></figure><h2>Отладчик</h2><h3>Упрощённая настройка панели инструментов в окне Debugger</h3><p>Теперь у вас есть возможность настроить панель инструментов отладчика так, чтобы она содержала именно те функции, которые вы используете ежедневно.</p><p>Щелкните правой кнопкой мыши рядом с kebab menu в верхней панели и выберите <em>Add to Debugger Toolbar</em>. Вы увидите список всех доступных действий – выберите те, которые подходят вашему проекту, и сделайте процесс отладки ещё удобнее.</p><figure class="full-width"><img data-src="https://habrastorage.org/getpro/habr/upload_files/a04/cc8/3d8/a04cc83d828a9de235eed068903f3b99.png" height="600" src="https://habrastorage.org/r/w1560/getpro/habr/upload_files/a04/cc8/3d8/a04cc83d828a9de235eed068903f3b99.png" width="1500"/></figure><h3>Текстовые всплывающие подсказки в inline-подсказках</h3><p>При отладке и просмотре значения с marked-up текстом теперь вы можете видеть его в форматированном виде, а не как просто длинную строку. Например, если значение представляет собой XML-данные для парсера, они будут отображаться в структурированном, удобочитаемом формате.</p><p>Ранее эта функция была доступна только для <em>watches</em>. Теперь мы распространили ту же функциональность на встроенную отладку, обеспечивая согласованность в обоих режимах.</p><figure class="full-width"><img data-src="https://habrastorage.org/getpro/habr/upload_files/657/e3c/886/657e3c88632128ab10644948fe4027c6.png" height="600" src="https://habrastorage.org/r/w1560/getpro/habr/upload_files/657/e3c/886/657e3c88632128ab10644948fe4027c6.png" width="1500"/></figure><p>Это все изменения, которые вошли в состав первой EAP-версии IntelliJ IDEA 2025.1. Ознакомьтесь с <a href="https://youtrack.jetbrains.com/articles/IDEA-A-2100662348/IntelliJ-IDEA-2025.1-EAP-1-251.14649.49-build-Release-Notes?_gl=1*7b7shz*_ga*NTE3NTcwNDczLjE3MzcwMTQ5ODY.*_ga_9J976DJZ68*MTczNzAxNDk4NS4xLjEuMTczNzAxNTEwMC4wLjAuMA.."><u>release notes</u></a>, чтобы увидеть полный список изменений.</p><figure class="full-width"><img data-src="https://habrastorage.org/getpro/habr/upload_files/fa1/d41/d1d/fa1d41d1d98c41cb20698af46cf54fe5.png" height="334" src="https://habrastorage.org/r/w1560/getpro/habr/upload_files/fa1/d41/d1d/fa1d41d1d98c41cb20698af46cf54fe5.png" width="1560"/></figure><p>Присоединяйтесь к русскоязычному сообществу разработчиков на Spring Boot в телеграм - <a href="https://t.me/+oBwACbkvJ19iODAy">Spring АйО</a>, чтобы быть в курсе последних новостей из мира разработки на Spring Boot и всего, что с ним связано.</p><p></p></div></div></div><!-- --><!-- --></div><!-- --><!-- --></div><!--]--><!-- --><div class="tm-article-presenter__meta" data-test-id="article-meta-links"><div class="tm-separated-list
