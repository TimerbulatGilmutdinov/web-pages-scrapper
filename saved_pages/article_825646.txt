представляет HTMX? <br/>Это легковесная библиотека, которая добавляет новые атрибуты для html. <br/>Нам не нужно больше парсить JSON.  <br/>Делаем запрос к серверу и вставляем кусок html кода на страницу. <br/>HTMX занял <strong>второе</strong> место в рейтинге самых популярных фронтенд фрейморков по версии <a href="https://risingstars.js.org/2023/en#section-framework" rel="noopener noreferrer nofollow">JavaScript Rising Stars 2023</a>.</p><p>Очевидно, что у такого подхода существуют свои ограничения и пока не всё получится сделать, но потенциал огромен. Новое золото!</p><div class="floating-image"><p></p></div><figure class="full-width"><img alt="Зумерки переизобрели AJAX и хвастаются. Ktor<br/>2. HTMX<br/>3. Tailwind для стилей<br/>4. Kotlinx.html как DSL для написания HTML разметки прямо в Kotlin коде.<br/>5. Sign in to save videos, subscribe, and more” или </em>другое похожее сообщение "если вы не бот, то авторизуйтесь"</p><p>Простейшее решение:</p><pre><code class="kotlin">fun create() : InnerTubeApiImpl {  
  return InnerTubeApiImpl(client = HttpClient(CIO)   {  
  
  
  engine { 
     // Меняем на свой адрес прокси 
     proxy = ProxyBuilder.http("http://0.0.0.0:8000/")  
  }  
  
  defaultRequest { 
        //Авторируемся через Username и password
        val credentials = Base64.getEncoder().encodeToString("USERNAME:PASSWORD".toByteArray())  
        header(HttpHeaders.ProxyAuthorization, "Basic $credentials")  
     }  
  })</code></pre><p>Спасибо за внимание.</p><p>Надеюсь, что эта статья была для вас полезной!</p><p></p></div></div></div><!-- --><!-- --></div><!-- --><!-- --></div><!--]--><!-- --><div
